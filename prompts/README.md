# Prompts 폴더

이 폴더는 AI 프롬프트와 마크다운 문서를 저장하는 곳입니다.

## 용도

- AI에게 전달할 프롬프트 템플릿
- 분석 계획 문서
- 프로젝트 메모
- 아이디어 정리

## 예시

```
prompts/
├── analysis_plan.md
├── prompt_template.md
├── ideas.md
└── README.md
```

## 사용 방법

마크다운 파일을 만들어서 프롬프트나 계획을 작성하세요.

### 예시: analysis_plan.md
```markdown
# 데이터 분석 계획

## 목표
- 데이터에서 패턴 찾기
- 시각화 생성

## 단계
1. 데이터 로드
2. 전처리
3. EDA
4. 모델링
```
