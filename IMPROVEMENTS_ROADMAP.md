# 🚀 Metropy 개선 로드맵

## 📊 추가하면 좋을 데이터셋

### 🔴 우선순위 높음 (즉시 효과)

#### 1. **실시간 혼잡도 데이터** ⭐⭐⭐⭐⭐
**출처**: 서울교통공사 실시간 열차 혼잡도 API
**효과**:
- 현재 시점의 정확한 혼잡도 반영
- 예측에서 실측으로 전환
- 사용자 신뢰도 대폭 상승

**구현 방법**:
```python
# 서울교통공사 Open API
endpoint = "http://swopenapi.seoul.go.kr/api/subway/{API_KEY}/json/realtimeStationArrival/..."
# 5분마다 업데이트, WebSocket으로 실시간 푸시
```

**예상 효과**: 추천 정확도 +30%

---

#### 2. **칸별 실제 혼잡도 데이터** ⭐⭐⭐⭐⭐
**출처**: 지하철 CCTV 영상 분석 (AI), 또는 차량 내부 센서
**현재 문제**: 역 전체 하차량만 알고, 칸별 분포는 추정치

**데이터 구조**:
```json
{
  "station": "강남",
  "time": "08:30",
  "direction": "내선",
  "car_congestion": {
    "1": 85,  // 혼잡도 퍼센티지
    "2": 92,
    "3": 95,  // 중간 칸이 가장 혼잡
    "4": 93,
    ...
  }
}
```

**활용**:
- SeatScore 검증 및 개선
- 머신러닝 모델 학습 데이터

**예상 효과**: 추천 정확도 +50%

---

#### 3. **날씨 데이터** ⭐⭐⭐⭐
**출처**: 기상청 Open API
**영향**:
- 비오는 날: 지하철 이용 +20~30%
- 눈오는 날: +40~50%
- 미세먼지 나쁨: +15~20%

**구현**:
```python
# 날씨에 따른 가중치 조정
if weather == "rain":
    alpha *= 1.2  # 혼잡도 20% 증가
elif weather == "snow":
    alpha *= 1.4
```

**추가 기능**:
- "오늘은 비가 와서 평소보다 혼잡할 것으로 예상됩니다" 알림

**예상 효과**: 추천 정확도 +15%

---

#### 4. **공휴일/이벤트 달력** ⭐⭐⭐⭐
**출처**: 공공데이터포털 공휴일 API
**영향**:
- 평일 vs 주말 패턴 차이
- 공휴일: 출퇴근 러시 없음
- 대형 이벤트 (콘서트, 야구 경기): 특정 역 혼잡

**데이터 예시**:
```json
{
  "date": "2025-05-05",
  "type": "holiday",
  "name": "어린이날",
  "pattern": "weekend"
},
{
  "date": "2025-06-15",
  "type": "event",
  "location": "잠실종합운동장",
  "name": "BTS 콘서트",
  "expected_traffic": 50000
}
```

**예상 효과**: 특수일 추천 정확도 +40%

### 🟡 우선순위 중간 (가치 높음)

#### 5. **역별 환승객 비율** ⭐⭐⭐
**출처**: 서울 열린데이터광장
**활용**:
- 환승역에서는 환승객이 특정 칸에 몰림
- 환승 동선을 고려한 추천

**예시**:
- 강남역 2→신분당선: 5-6호차에서 환승 편리
- 교대역 2→3호선: 7-8호차 추천

---

#### 6. **노약자석/임산부석 위치** ⭐⭐⭐
**활용**:
- 사용자 프로필에 따른 맞춤 추천
- "노약자석 우선 추천" 옵션

---

#### 7. **칸별 온도 데이터** ⭐⭐
**출처**: 차량 내부 온도 센서 (있다면)
**활용**:
- 여름: 에어컨 세기
- 겨울: 히터 위치
- "시원한 칸" vs "따뜻한 칸" 추천

---

#### 8. **역별 주변 POI 데이터** ⭐⭐⭐
**출처**: 카카오/네이버 지도 API
**활용**:
- 출근 목적지 추정 (업무 지구, 학교)
- 퇴근 목적지 추정 (주거 지역)
- 맥락 인식 추천

---

#### 9. **사용자 피드백 데이터** ⭐⭐⭐⭐
**수집 방법**: 앱 내 "추천이 정확했나요?" 투표
**활용**:
- 모델 개선을 위한 실제 ground truth
- A/B 테스팅

**구조**:
```json
{
  "user_id": "hash_value",
  "route": "강남→시청",
  "recommended_car": 3,
  "user_rating": 5,
  "seated": true,
  "timestamp": "2025-01-15T08:30:00"
}
```
